# WebCrypto
Simple Flask basked web crypt.

## How to install ##
1. Create a virtual env.
```bash
virtualenv .env
source .env/bin/activate
```

2. Run the Flask server
```bash
python index.py
```

3. Open browser at http://localhost:5000
4. Profit.
